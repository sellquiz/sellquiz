Deklarationen

Schreiben Sie einen Codeausschnitt, in dem zwei Variablen wie folgt definiert werden: Eine Variable mit dem Namen "x" speichert eine Gleitkommazahl und weist dieser den Wert 1.23 zu. Eine weitere Variable mit dem Namen "text" soll die Zeichenkette "hallo" beinhalten.

	prog := JavaBlock {
		assert x == 1.23;
		assert text.equals("hallo");
	}

$ #prog $

_Hinweis: schreiben Sie nur die Deklarationen selbst. Geben Sie keine Klasse und keinen Methodenkopf an!_
